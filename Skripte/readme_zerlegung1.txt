1- Erstmal die Datei zerlegung.py herunterladen
2- Dann mit der folgenden Kommandozeile ausführen:
	python zerlegung.py
3- Dann den Datenordner und den Zielordner eingeben
4- Dann wird die zerlegung durchgeführt
5- Die Datei zuordnung_zieldaten.py herunterladen
6- Unter der Bedingung, dass man als Zielordner bei der Ausführung des Skripts zerlegung.py den Ordner angegeben hat, in dem man gerade arbeitet, den Befehl python zuordnung_zieldaten.py ausführen // Das Skript funktioniert für alle VP außer VP_131 (den habe ich vorher entfernt)
